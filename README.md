
# AI GYM TRACKER

Building a Real-Time AI Pose Estimation for GYM Tracker using Mediapipe.

## Run Locally

Clone the project

```bash
  git clone https://github.com/brianadit24/AIGymTracker
```

Go to the project directory

```bash
  cd AIGymTracker
```

Install requirements

```bash
  pip install -r requirements.txt
```

Run the Model

```bash
  python3 model.py
```
